
# Перевод курсов Excel и Веб-разработки на русский язык

## Что меняется

Полная замена английских текстов на русские описания для двух курсов в `src/data/courses.ts`, основанная на предоставленном пользователем контенте.

## Файл: `src/data/courses.ts`

### Курс Excel (строки 178-289)

**Общая информация:**
- `tagline`: "Анализ, отчёты и автоматизация"
- `description`: "Освойте Excel как профессиональный инструмент для анализа данных, построения отчётов и автоматизации рабочих процессов..."

**Benefits (4 шт.):**
1. Системный подход -- от базовых инструментов до продвинутой аналитики
2. Практика на реальных задачах -- работа с таблицами, отчётами и кейсами
3. Минимум теории -- максимум применения
4. Готовые шаблоны и примеры для работы

**Audience (4 шт.):**
1. Начинающие пользователи -- освоить Excel с нуля
2. Специалисты по отчётам и данным (аналитики, экономисты, бухгалтеры, маркетологи)
3. Студенты и выпускники -- практические навыки анализа данных
4. Все, кто хочет повысить эффективность работы

**Requirements (4 шт.):**
- Базовые навыки работы с компьютером
- Установленный Microsoft Excel (2016+)
- Умение работать с файлами и папками
- Желание развивать навыки анализа данных

**Curriculum (5 модулей на русском):**
1. Основы Excel -- интерфейс, ввод данных, базовые формулы
2. Формулы и функции -- IF, SUMIF, ВПР, текстовые и логические функции
3. Визуализация данных -- диаграммы, условное форматирование, дашборды
4. Сводные таблицы -- создание, группировка, вычисляемые поля
5. Автоматизация -- Power Query, Power Pivot, макросы

**FAQ (4 шт. на русском)**

### Курс Веб-разработки (строки 291-404)

**Общая информация:**
- `tagline`: "Создание сайтов на CMS: WordPress, Wagtail, Ghost"
- `description`: "Освойте создание и управление веб-сайтами с использованием современных CMS..."

**Benefits (4 шт.):**
1. Практическая направленность -- создание собственного веб-проекта
2. Работа с несколькими CMS -- WordPress, Wagtail, Ghost
3. Современный стек -- Tailwind CSS, AI-инструменты
4. Навыки для фриланса и коммерческой разработки

**Audience (4 шт.):**
1. Новички в веб-разработке
2. Веб-дизайнеры, расширяющие технические навыки
3. Студенты и специалисты, обновляющие стек
4. Фрилансеры, работающие с сайтами на CMS

**Requirements (4 шт.):**
- Базовые знания HTML и CSS
- Желательно базовое понимание Python
- Компьютер с доступом к интернету
- Желание учиться и практиковаться

**Curriculum (5 модулей на русском):**
1. Основы веб-разработки -- домены, хостинг, архитектура CMS
2. WordPress -- установка, настройка, темы и плагины
3. Wagtail (Django CMS) -- разработка на Python, шаблоны
4. Ghost -- настройка, контент, интеграции
5. Tailwind CSS и AI -- современная вёрстка, GPT-инструменты
