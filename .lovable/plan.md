

# Удаление Google OAuth из админ-панели

## Что изменится

Из формы входа в админ-панель (`src/pages/Admin.tsx`) будет удалена кнопка "Войти через Google" и разделитель "или". Останется только вход по email/паролю.

## Технические детали

### Файл: `src/pages/Admin.tsx`

- Удалить импорт `lovable` (строка 10)
- Удалить блок разделителя "или" (строки 63-66)
- Удалить кнопку Google OAuth (строки 67-77)

Форма входа станет компактнее: только поля Email, Пароль и кнопка "Войти".
